# Busca de Peças (PWA iPhone) — versão com seus dados
Como instalar no iPhone:
1) Publique esta pasta em HTTPS (GitHub Pages, Netlify ou Vercel).
2) No iPhone (Safari), abra a URL.
3) Toque em **Compartilhar** → **Adicionar à Tela de Início**.
4) Abra o ícone instalado — funciona offline.

Se quiser atualizar os dados depois:
- Substitua o arquivo **dados.json** com o novo export da planilha.
